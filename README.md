# ActiTimeProject
A Github repository for Functional and UI Automation Testing using Java, Selenium, TestNG and Appium. This is a Test Automation Framework used to Automate Scenarios for ActiTime Application. 

## Pre-requisites
**Windows OS**  
**Eclipse IDE**  
**JDK 7 or JDK 8**  
**MAVEN**  
**APPIUM**  
**PDANET**
