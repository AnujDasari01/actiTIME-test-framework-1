# ActiTimeProject
A Github repository for Functional and UI Automation Testing using Java, Selenium, TestNG and Appium. 

## Pre-requisites
* Windows OS  
* Eclipse IDE  
* JDK 7 or above  
* Maven  
* Appium  
* PDAnet  

## Let's get started
*Download the project from*  
https://github.com/AnujDasari/ActiTimeProject

## To run from commandline
`mvn exec:java`
